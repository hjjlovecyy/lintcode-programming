Flatten List
-----
Description:

    Given a list, each element in the list can be a list or integer. flatten it into a simply list with integers.
    
Example:
    
    Given [1,2,[1,2]], return [1,2,1,2].    
    Given [4,[3,[2,[1]]]], return [4,3,2,1].
    
Challenge:
    
    Do it in non-recursive.
